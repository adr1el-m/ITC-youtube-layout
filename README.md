Welcome to **PUP MSC's Frontend Web Development Training Repository**, where love at first click is just the beginning! Whether you're rekindling your passion for HTML, having a fling with CSS, or diving deep into a committed relationship with JavaScript, this repository is here to make your coding journey unforgettable. 💘

---

## Overview

This repository is designed for hands-on learning of frontend development fundamentals using **HTML**, **CSS**, and **JavaScript**. Each section includes detailed documentation, practical exercises, and quizzes to reinforce your understanding. 

### Key Features:
- **Documentation**: Learn the foundational concepts step-by-step.
- **Exercises**: Apply your knowledge with practical activities.
- **Knowledge Check**: Validate your understanding with quizzes (available via the provided link).

---

## Lab Topics and Exercises

### 1. HTML Lab
- **Key Topics**: Semantic structure, headings, lists, links, and images.
- **Exercises**:
  1. Simple Profile Page
  2. Product Listing Page
  3. [Add your own exercises]

### 2. CSS Lab
- **Key Topics**: Selectors, properties, box model, and flexbox.
- **Exercises**:
  1. Styled Card Design
  2. Flexbox Layout
  3. [Add your own exercises]

### 3. JavaScript Basics Lab
- **Key Topics**: Variables, data types, operators, and control structures.
- **Exercises**:
  1. Basic Calculator
  2. Todo List
  3. [Add your own exercises]

### 4. JavaScript Advanced Lab
- **Key Topics**: DOM manipulation and events.
- **Exercises**:
  1. Interactive Image Gallery
  2. Dynamic Form
  3. [Add your own exercises]

## How to Use This Repository

1. **Navigate** to the topic folder (e.g., `HTML/`, `CSS/`).
2. **Read the Documentation** in the `Documentation/` folder to understand the concepts.
3. **Complete the Exercises** in the `Exercises/` folder.
4. **Submit Your Work**: Follow submission guidelines shared via the provided forms.
