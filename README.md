## 📋 About This Project

This project is a **YouTube homepage layout clone** created as an activity for **Introduction to Computing (ITC)** at the **Polytechnic University of the Philippines** under the guidance of **Ma'am Monina Baretto**.

The goal of this activity was to practice and demonstrate front-end web development skills by recreating the familiar layout of YouTube. Instead of using placeholder content, I personalized it by featuring **my favorite YouTube videos as of January 2025**.

## ✨ Features

- 📺 Responsive YouTube-style video grid layout
- 🎨 Clean and modern UI design
- 🖼️ Video thumbnails with duration overlays
- 📝 Video titles, channel names, and metadata display
- ✅ Verified channel badges
- 🌙 Dark theme inspired by YouTube's interface

---

## 🛠️ Built With

- **HTML5** - Semantic markup structure
- **CSS3** - Styling and layout (Flexbox/Grid)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/ITC-youtube-layout.git
   ```

2. **Open the project**
   ```bash
   cd ITC-youtube-layout
   ```

3. **Launch the site**
   - Open `index.html` in your browser, or
   - Use a live server extension in VS Code
